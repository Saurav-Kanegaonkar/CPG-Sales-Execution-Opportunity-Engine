# Data Sources

Synthetic source-style data for CPG sales execution analytics.

- `entities.csv`: 36 sales priority records.
- `daily_metrics.csv`: 5,040 sales priority-day metric rows.
- `source_events.csv`: 760 operational events, QA checks, experiments, and stakeholder asks.
- `recommended_actions.csv`: 220 candidate actions with effort and expected lift.
